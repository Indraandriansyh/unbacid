# Universitas Nusa Bangsa (UNB) Website

Full-stack university website for Universitas Nusa Bangsa (UNB), Bogor, Indonesia.

## Architecture

**Monorepo** managed with `pnpm workspaces`.

### Packages
- `unb/artifacts/unb-website` — React/Vite frontend (port 25692)
- `unb/artifacts/api-server` — Express API server (port 8080)
- `unb/lib/db` — Drizzle ORM + PostgreSQL schema
- `unb/lib/api-zod` — Zod validation schemas (generated by orval)
- `unb/lib/api-client-react` — TanStack Query hooks (generated by orval)
- `unb/scripts` — Seed scripts

### Tech Stack
- **Frontend**: React 18, Vite, TypeScript, TailwindCSS, shadcn/ui, TanStack Query
- **Backend**: Express.js, TypeScript
- **Database**: PostgreSQL via Drizzle ORM
- **API contract**: OpenAPI → orval → Zod + React Query hooks

## Key Features

### Public Website
- Multi-language support (ID/EN/SU/JV)
- Dark/light mode
- **Beranda** — Hero, banners, stats, CTA
- **Profil** — About university, vision/mission, structure
- **Berita** — News articles
- **Fakultas & Prodi** — Faculty + Program pages
- **LPPM** — Research, community service, innovation, HAKI, books, seminar tabs
- **Pendaftaran** — Admission page with:
  - Alur, Jalur, Waktu, Biaya Pendidikan sections
  - Online registration form
  - **Payment Modal** (Midtrans Snap or Bank Transfer BJB Syariah)

### Admin Dashboard (`/admin`)
Login with password stored in localStorage (key: `admin_authenticated`).

Menus:
- **Home** — Edit hero text, banners, grid, CTA
- **Profile** — Edit profile page content
- **Data Dosen** — Manage lecturers
- **Berita** — Manage news
- **LPPM** — Manage LPPM content
- **Pendaftaran** — Manage admission page (7 tabs: Banner, Alur, Jalur, Waktu, Biaya, Formulir, **Pengaturan Bayar**)
- **Data Pendaftar** — View all registrants table with payment status management
- **Fakultas** — Per-faculty content management
- **Program Studi** — Per-prodi content management

### `site_settings`
Key-value store. Used keys:
- `homeContent` — Home page content
- `profileContent` — Profile page content
- `newsContent` — News content
- `registrationContent` — Admission page content (banners, flow, tracks, waves, tuition, brochureUrl, formUrl, whatsappNumber)
- `paymentSettings` — Payment config (midtransEnabled, midtransClientKey, midtransIsProduction, bankName, bankAccountNumber, bankAccountName, registrationFee)
- `lppmContent` — LPPM page content
- `facilitiesContent`, `dosenData`, `fakultasContent.*`, `prodiContent.*`

## Payment System

### Midtrans Snap
- Server key: env var `MIDTRANS_SERVER_KEY`
- Client key: stored in `paymentSettings.midtransClientKey`
- Production mode: env var `MIDTRANS_IS_PRODUCTION=true` (or from `paymentSettings.midtransIsProduction`)
- Webhook: `POST /api/registrations/midtrans-notify`

### Bank Transfer
- BJB Syariah: 0040102001205 an. Universitas Nusa Bangsa
- Confirmation via WhatsApp to admin
- Admin manually changes `payment_status` in Data Pendaftar

## API Endpoints

- `GET /api/site-settings` — All settings
- `POST /api/site-settings` — Update/create setting
- `GET /api/registrations` — List all registrations (admin)
- `POST /api/registrations` — Submit new registration
- `POST /api/registrations/:id/payment-init` — Init payment (midtrans or bank_transfer)
- `POST /api/registrations/midtrans-notify` — Midtrans webhook
- `PATCH /api/registrations/:id/payment-status` — Admin manual status update
- `GET /api/uploads/:filename` — Serve uploaded files

## Running Locally
```bash
# Install deps
pnpm install

# Start frontend
pnpm --filter @workspace/unb-website run dev

# Start API
pnpm --filter @workspace/api-server run dev
```
